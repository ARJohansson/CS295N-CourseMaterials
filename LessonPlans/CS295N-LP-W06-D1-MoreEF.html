Seeding the Database

Seed data is data you add to the database when it is first created. This is usually data that you will use to test the web application

How to add seed data

  1. Add a static class named SeedData.
    • This class will contain a static method with this signature: public static void Initialize(IServiceProvider serviceProvider)
    • In the Initialize method:
      • Create an instance of your class that is a subclass of DbContext
      • Check to see if there is already seed data in your database, if there is, return.
      • Seed the database using the AddRange method on your context object's collection of model objects
  2. Add code to Program.Main to call the SeedData.Initialize method.
  3. Test it! You should see the movies displayed on the index page now.
The steps above were based on Working with SQL Server LocalDB and ASP.NET Core - Note that the section of this tutorial on seed data will work for VS Code on Mac OS with no changes

Searching the Database

In the code behind for the page on which you wish to do a search, add code to search the database using a LINQ statement like:
movies = movies.Where(s => s.Title.Contains(searchString));

Based on Adding search to an ASP.NET Core MVC app
